Zodiac API

Этот API предоставляет информацию о знаках зодиака. Вы можете использовать следующие эндпоинты:

Текущий знак зодиака

GET /api/current-sign

Возвращает знак зодиака, соответствующий текущей дате.

{"sign": "Водолей"}

Знак зодиака по дате

GET /api/sign-by-date?date=MM-DD

Возвращает знак зодиака, соответствующий переданной дате.

{"sign": "Скорпион"}

Даты по знаку зодиака

GET /api/dates-by-sign?sign=Знак

Возвращает диапазон дат, соответствующий переданному знаку зодиака.

{
  "sign": "Овен",
  "dates": [
    "03-21",
    "04-19"
  ]
}
